Што се Кориснички Функции во C++?

Во C++, корисничките функции се функции дефинирани од корисникот, кои му овозможуваат на програмерот да ја подели програмата на помали, управливи делови. Тие се користат за организирање и повторно користење на кодот, што го прави кодот почист, полесен за одржување и разбирлив.

Пример за корисничка функција


#include <iostream>
using namespace std;

int sobiranje(int a, int b) {
    return a + b; // Враќа збирот на a и b
}

int main() {
    int broj1, broj2;
    
    cout << "Внесете два броја: ";
    cin >> broj1 >> broj2;
    
    int rezultat = sobiranje(broj1, broj2);
    cout << "Збирот е: " << rezultat << endl;
    
    return 0;
}

Објаснување на примерот

  • Функција Собирање: Оваа функција прима два целобројни аргументи и враќа целобројна вредност. Во телото на функцијата, таа го пресметува збирот на a и b и ја враќа оваа вредност.
  • Повик на функцијата: Во main() функцијата, корисникот ги внесува два броја, кои потоа се проследуваат на sobiranje функцијата, и резултатот се прикажува.
